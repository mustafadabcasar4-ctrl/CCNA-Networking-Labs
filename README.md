# CCNA-Networking-Labs

A collection of hands-on Cisco networking labs covering switching, routing, network services, security, and troubleshooting.

## About This Repository

This repository documents my practical networking experience through simulated enterprise network environments using Cisco technologies.

Each lab includes:
- Network topology diagrams
- Device configurations
- Verification commands
- Troubleshooting notes
- Lessons learned

## Technologies & Tools

- Cisco IOS
- Cisco Packet Tracer
- GNS3 / EVE-NG
- Wireshark
- Linux networking tools

## Networking Topics Covered

### Switching
- VLAN configuration
- Access and trunk ports
- Inter-VLAN routing
- Spanning Tree Protocol (STP)
- EtherChannel

### Routing
- IPv4 addressing and subnetting
- Static routing
- OSPF
- Default routes
- Routing troubleshooting

### Network Services
- DHCP configuration
- NAT/PAT
- DNS fundamentals
- NTP

### Network Security
- Access Control Lists (ACLs)
- Port security
- SSH configuration
- Basic firewall concepts

### Troubleshooting
- Connectivity issues
- VLAN problems
- Routing failures
- Packet analysis using Wireshark
